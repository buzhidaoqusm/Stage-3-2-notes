## Some Laws
用得不多
{P} S {Q} ∧ {P} S {R} ⇒ {P} S {Q ∧ R}
{P} S {Q} ∧ {R} S {Q} ⇒ {P ∨ R} S {Q}
{P} S {Q} ∧ (Q ⇒ R) ⇒ {P} S {R}         “weakening/strengthening post”
- 从左往右叫weakening，反之为strengthening
{P} S {Q} ∧ (R ⇒ P) ⇒ {R} S {Q}         “weakening/strengthening pre”
{P} S0 {Q} ∧ {Q} S1 {R} ⇒ {P} S0;S1 {R} “concatenation”

## Weakest Precondition
```
(0) { 11 ≤ x } x := x + 1 { 5 ≤ x }
(1) { 10 ≤ x } x := x + 1 { 5 ≤ x }
(2) { 9 ≤ x } x := x + 1 { 5 ≤ x }
(3) { 8 ≤ x } x := x + 1 { 5 ≤ x }
(4) { 7 ≤ x } x := x + 1 { 5 ≤ x }
(5) { 6 ≤ x } x := x + 1 { 5 ≤ x }
(6) { 5 ≤ x } x := x + 1 { 5 ≤ x }
(7) { 4 ≤ x } x := x + 1 { 5 ≤ x }
(8) { 3 ≤ x } x := x + 1 { 5 ≤ x }
```
{11 ≤ x} ⇒ {10 ≤ x} ⇒ ……⇒ {4 ≤ x}
从左往右逐渐weaker
 {11 ≤ x} is the strongest and {4 ≤ x} is the weakest
 {4 ≤ x} is the Weakest Precondition guarantees doing x:=x+1 achieves 5 <=x
WP.(x := x + 1).(5 ≤ x) ≡ 4<= x
WP.S.Q
