# 定义
Software Project Management is a **system management method** based on software project, which uses the relevant knowledge, techniques and tools for **planning, organizing, advising and controlling** each stage of software project cycle to achieve the project objectives.

# Project and Project Management
## Project

> Project定义

A project is a **temporary** endeavor undertaken to create a **unique product**, service, or result
- temporary: a project has a **definite beginning** and **end**.
**Repetitive** elements **may be present** insome project deliverables, **not change** the **unique** characteristicsof the project

> Project产物

- **product**
	- **component, enhancement,  end item** of another item
- **service** or a capability
- **improvement** in the existing product
- **result**
	- outcome or document

> Exmaple

**Product** : bridge, building 
**Service**:    training 
**Result** :     research
